自定义样式 ​

名称说明其他
data-jds25-class=""实现自定义样式
Modal弹出层 自定义样式

文字标题文字标题

文字文字文字文字文字文字文字文字文字文字文字文字文字文字文字

html
<span 
    data-jds25="jdsModal" 
    data-jds25-class="dp"
>
    Modal弹出层 自定义样式
</span>
<div data-jds25-body>
    <h3>文字标题文字标题</h3>
    <img src="./img/dummy-half.png" alt="">
    <p>文字文字文字文字文字文字文字文字文字文字文字文字文字文字文字</p>
</div>
scss
[data-jds25=modal-wrap].dp{
    background: #1677ff;
    color: #fff;
    left: calc(50% - ((100vw - (32px * 2)) / 2));// [data-jds25=wrap] width: calc(100vw - (32px * 2));
    top: calc((50% - (100vh / 2)) + (32px * 1));// [data-jds25=modal-body] max-height: calc(100vh - 32px - (32px * 2));
    width: calc(100vw - (32px * 2));
    [data-jds25=modal-body]{
        height: calc(100vh - 32px - (32px * 2));
        max-height: calc(100vh - 32px - (32px * 2));
    }
    .close{
        color: #fff;
        right: -16px;
        top: -16px;
        background: #1677ff;
        border-radius: 50%;
        box-shadow: 0 3px 6px -4px rgba(0, 0, 0, 0.12),
            0 6px 16px 0 rgba(0, 0, 0, 0.08), 0 9px 28px 8px rgba(0, 0, 0, 0.05);
    }
}
[data-jds25=modal-mask].dp{
    transition: all .1s ease-in-out;
}
[data-jds25=modal-wrap].dp{
    transition: all .2s ease-in-out;
}

自定义样式 底部弹出FULL屏 ​

Modal底部弹出全屏

文字标题文字标题

文字文字文字文字文字文字文字文字文字文字文字文字文字文字文字

html
<span 
    data-jds25="jdsModal" 
    data-jds25-class="fullup"
>
    Modal底部弹出全屏
</span>
<div data-jds25-body>
    <h3>文字标题文字标题</h3>
    <img src="./img/dummy-half.png" alt="">
    <p>文字文字文字文字文字文字文字文字文字文字文字文字文字文字文字</p>
    <button onclick="JDS25Tools.trigger(document.querySelector('[data-jds25=modal-mask]'))" class="pr">CLOSE</button>
</div>
scss
[data-jds25=modal-wrap].fullup{
    transition: all .3s ease-in-out;
    left: calc(50% - ((100vw - (0px * 2)) / 2));
    top: 100vh;
    width: calc(100vw - (0px * 2));
    [data-jds25=modal-body]{
        height: calc(100vh - 32px - (0px * 2));
        max-height: calc(100vh - 32px - (0px * 2));
    }
    &.ani{
        top: calc((50% - (100vh / 2)) + (0px * 1));
    }
}

    data-jds25="jdsModal" 
    data-jds25-class="halfup"
>
    Modal底部弹出半屏
</span>
<div data-jds25-body>
    <h3>文字标题文字标题</h3>
    <img src="./img/dummy-half.png" alt="">
    <p>文字文字文字文字文字文字文字文字文字文字文字文字文字文字文字</p>

[data-jds25=modal-wrap].halfup{
    transition: all .2s ease-in-out;
    left: calc(50% - ((100vw - (0px * 2)) / 2));
    top: initial;
    bottom: calc((50vh - 32px - (0px * 2)) * -1);
    width: calc(100vw - (0px * 2));
    [data-jds25=modal-body]{
        height: calc(50vh - 32px - (0px * 2));
        max-height: calc(50vh - 32px - (0px * 2));
    }
    &.ani{
        bottom: 0;
    }
}
